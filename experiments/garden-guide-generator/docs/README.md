# Garden Guide Generator - Experiment Documentation

## Overview

Garden Guide Generator is a location-based printable seed guide tool that generates accessible, scannable PDF guides for any zip code showing native plants, adapted varieties, invasive warnings, and where to buy them.

## Experiment Statement

I'm attempting to build Garden Guide Generator, a location-based printable seed guide tool that generates accessible, scannable PDF guides for any zip code showing native plants, adapted varieties, invasive warnings, and where to buy them, combining location-based discovery with digital-to-physical transformation and ADHD-friendly formatting to make gardening more accessible.

## Combined Strengths

This experiment combines the best aspects of three existing experiments:

1. **From Seed Finder**: Location-based seed discovery, native/adapted/invasive status tracking, SEO/public discovery pages
2. **From Photo Memories**: Digital-to-physical transformation, PDF generation, print-on-demand capabilities
3. **From Best Day Ever**: ADHD-friendly formatting, scannable layouts, printable design principles
