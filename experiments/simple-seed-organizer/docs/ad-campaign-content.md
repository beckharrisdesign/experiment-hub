## Testing Strategy

### What to Test

1. **Headlines**: Which pain point resonates most?

   - Rebuying seeds
   - Seed viability
   - Organization
   - Simplicity

2. **Images**: Which visuals perform best?

   - Before/after
   - App mockups
   - Real seed packets
   - Text overlays

3. **CTAs**: Which call-to-action converts?

   - "Learn More"
   - "Get Early Access"
   - "Try It Simple"
   - "Organize My Seeds"

4. **Platforms**: Which platform performs better?
   - Meta (Facebook/Instagram)
   - Pinterest
   - (Could test Google Ads later)

### Success Metrics

**Primary Metrics**:

- CTR (Click-Through Rate): Target > 2%
- Landing page conversion: Target > 10% (visitors → form submission)
- Cost per click (CPC): Track, optimize for < $1-2
- Cost per signup: Track, optimize for < $10-20

**Secondary Metrics**:

- Engagement rate (likes, comments, shares)
- Time on landing page
- Bounce rate
- Form start rate vs. completion rate

### Optimization Rules

**After 2-3 Days**:

- Pause ads with CTR < 1%
- Increase budget for ads with CTR > 3%
- Test new variants for low performers

**After 1 Week**:

- Double down on top 2-3 performers
- Pause all low performers
- Test new angles based on learnings

**After 2 Weeks**:

- Evaluate overall campaign performance
- Decide: Proceed to MVP or pivot messaging
- Document learnings for future campaigns

## Landing Page Match

**Important**: Ad content should match landing page messaging.

- If ad says "Stop rebuying seeds", landing page should lead with that problem
- If ad says "Simple, not complex", landing page should emphasize simplicity
- Consider creating landing page variants to match ad angles (A/B test)

**Ad → Landing Page Flow**:

1. User clicks ad (sees specific angle)
2. Lands on landing page (matches that angle)
3. Sees consistent messaging
4. Clicks CTA (feels understood)
5. Fills form (commits to early access)

## Tracking Setup

### UTM Parameters

**Format**: `?utm_source=meta&utm_medium=social&utm_campaign=validation&utm_content=variant1`

**Parameters**:

- `utm_source`: meta, pinterest, google
- `utm_medium`: social, search, display
- `utm_campaign`: validation-test, validation-test-v2
- `utm_content`: stop-rebuying, seed-viability, messy-box, etc.

**Example URLs**:

- `https://simpleseedorganizer.com/?utm_source=meta&utm_medium=social&utm_campaign=validation&utm_content=stop-rebuying`
- `https://simpleseedorganizer.com/?utm_source=pinterest&utm_medium=social&utm_campaign=validation&utm_content=seed-viability`
