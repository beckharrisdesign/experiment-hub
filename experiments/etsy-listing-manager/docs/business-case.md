# Etsy Patternator
 
## Hypothesis
 
A workflow tool that generates SEO-optimized listings, product images, and customer messages from a single artwork file will cut per-listing time from 30–60 minutes to under 15 — so artists spend more time making and less time selling.
 
---
 
## Why This Matters to Me
 
I've watched my mother run her Etsy shop for years — and every time she gets a new laptop, she has to re-learn her own business from scratch.
 
---
 
## Who It's For
 
Independent artists — illustrators, painters, printmakers, and pattern designers — who sell digital or physical work on Etsy and want to spend their time creating, not formatting listings. Craft pattern creators (embroidery, sewing, quilting) are a natural fit from day one; fine art sellers are a close second.
 
---
 
## What It Does
 
- Define brand voice and creative direction once; apply it automatically across every listing, image, and message
- Track work from concept to listed on a simple kanban
- Generate listing title (140 chars, SEO-optimized), description, and 13 keyword tags from an artwork name and basic details
- Import any artwork file and generate mockups, lay flats, store header, profile image, and customer downloads — all to Etsy spec
- Generate order confirmations, download instructions, follow-up messages, and FAQ responses in the store's voice
- Flag missing keywords across title, description, and tags
Not in MVP: Etsy API integration, cloud sync, multi-user features, social scheduling, direct publishing.

## Existing Options
 
| Product | Price | User Base | Strength | Limitation |
|---|---|---|---|---|
| Marmalead | $19/mo | ~260K subscribers | Keyword research depth | SEO only, not workflow automation |
| eRank | Free–$30/mo | ~2M users | Large user base, analytics | SEO only, same niche gap |
| Listings Plus | $15–30/mo | est. 2K–5K subscribers | Bulk listing management | Generic, no image or content generation |
| Canva | Free–$15/mo | ~185M users | Image quality | Manual, not connected to listing workflow |
 
No existing tool combines listing authoring, image generation, and customer communications in one workflow built for independent artists and craft creators.
 
---
 
## Biggest Unknown
 
Whether the generated image quality is good enough to use without manual editing — because if every image needs Canva cleanup anyway, the tool doesn't close the time gap.
 
---
 
## Validation Plan
 
| | Detail |
|---|---|
| Method | Self-timed use on real Watermark & Hue listings; log time per listing for 3 months |
| Traffic | N/A — personal tool, no external users in Year 1 |
| Budget | Claude API + image gen costs; est. $20–50/mo |
| Success | ≤15 min per listing sustained over 10+ listings |
| Decision point | If average stays above 20 min or images require significant rework, revisit image pipeline before any SaaS work |
 
The prototype is active and in personal use; early questions are around image quality and which step consumes the most time.

## Recommendation
 
GO on the personal tool — the time savings are real enough to measure and the prototype is already running. The evidence for a SaaS path is solid (validated market, clear niche gap, reasonable build cost), but nothing should move toward productization until self-timed data from 10+ listings confirms the 15-minute target is achievable. The next concrete step is structured logging: time each listing phase so you know whether the bottleneck is image quality, copy review, or the copy/paste export step.
