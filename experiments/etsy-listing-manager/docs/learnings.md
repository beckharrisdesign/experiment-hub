## Tool built, in personal use

The prototype is active. Key things to test and capture from real usage:

- Is 10–15 minutes per listing actually achievable, or is there still too much manual review/editing?
- Which part of the workflow takes the most time — image generation, listing text, or the export/copy-paste step?
- Is the brand identity setup actually paying off in consistency, or does it need more refinement per listing anyway?
- Is the generated image quality good enough to use, or does it need significant manual editing before it's listing-ready?
- What's the first thing you'd cut if you had to simplify the tool?
