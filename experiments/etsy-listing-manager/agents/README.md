# Shop Manager Agents

This directory contains specialized agents for the Shop Manager prototype. These agents provide domain-specific intelligence and automation for embroidery pattern store management.

## Available Agents

### Retail Advisor (`retail-advisor.md`)
**Purpose**: Suggests timely embroidery pattern themes based on trends, holidays, seasons, and cultural moments.

**Use Cases**:
- "What patterns should I create for the next 3 months?"
- "What's trending for spring embroidery patterns?"
- "What holiday patterns should I prepare for Halloween?"
- "Suggest patterns aligned with my brand identity for Q2"

**Key Features**:
- Calendar-based pattern suggestions
- Seasonal trend analysis
- Holiday and cultural event opportunities
- Market timing recommendations
- Etsy trend insights

## How to Use Agents

Agents can be invoked through:
1. **Direct API calls** (future implementation)
2. **Manual review** - Read agent instructions and apply insights
3. **Integration with Shop Manager features** - Agents inform Product Planning, Listing Authoring, etc.

## Adding New Agents

To create a new agent:
1. Create a new `.md` file in this directory
2. Follow the structure of existing agents:
   - Role definition
   - Purpose and workflow
   - Input/Output specifications
   - Step-by-step instructions
   - Integration points
3. Document how the agent integrates with Shop Manager features
