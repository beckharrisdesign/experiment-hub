Paint-by-number comparison

Pick a sample from assets/input/ or upload a file, then Run comparison. One click runs three full pipelines (A, B, C): SLIC segmentation plus region merges (defaults cap the long edge at 1600px before processing).

How long? Tiny images can finish in a few seconds. A typical phone photo after that downscale often lands around 30 seconds to a few minutes on a laptop — it varies with CPU and image complexity.

Hanging vs slow: While the blue banner’s Elapsed time keeps ticking, your browser is still waiting on the server for that one HTTP request (that is normal, not a frozen tab). It does not prove Python is making progress inside each variant. On macOS, open Activity Monitor → search Python → if CPU stays high for most of the wait, work is usually happening. If Elapsed passes ~10 minutes and Python’s CPU is ~0% the whole time, assume something is stuck and stop the request (close the tab or refresh).
