Figma Grabber

Capture any element.
Paste it into Figma.

A one-click bookmarklet that captures the layout or screenshot of any element on any page and copies a Figma-ready JSON payload to your clipboard. No login. No extension. Nothing leaves your browser.

Drag this to your bookmarks bar:

(or right-click → Bookmark this link)

Figma Grab

If dragging doesn't work, right-click the button → Bookmark this link

How to install

  1. 1

    Show your bookmarks bar

    In Chrome: ⌘ Shift B on Mac, or Ctrl Shift B on Windows/Linux.

  2. 2

    Drag the button to your bookmarks bar

    Click and hold Figma Grab above, then drag it up to the bookmarks bar. Release when you see the insertion point.

  3. 3

    You're done

    The bookmarklet is saved. You won't need to come back here.

How to use it

  1. 1

    Navigate to any page

    Open the site or UI you want to capture.

  2. 2

    Click Figma Grab in your bookmarks bar

    A toolbar appears at the bottom of the page with three options: Copy to clipboard, Entire screen, and Select element.

  3. 3

    Pick what to capture

    Click Select element to hover and click a specific component, or Entire screen for the full viewport, or Copy to clipboard to grab the whole page immediately.

  4. 4

    Paste the payload into your Figma MCP workflow

    A JSON payload is now on your clipboard. Paste it wherever your Figma MCP session expects design input.

Privacy

  • ✓ No Figma API calls. The bookmarklet never authenticates with Figma or touches any Figma endpoint.
  • ✓ Your clipboard is local. The JSON payload is written to your OS clipboard — nothing is stored server-side.
  • ✓ Works with your own credentials. Anyone with Figma edit access can use this — it requires no shared login.
  • ✓ The only network call is fetching the loader script from labs.beckharrisdesign.com — the same origin as this page.
